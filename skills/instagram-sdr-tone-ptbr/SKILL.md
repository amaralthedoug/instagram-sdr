# Instagram SDR Tone PT-BR

## Objetivo
Aplicar um método curto para construir mensagens que convertam no direct, preservando naturalidade brasileira e clareza comercial — independente do nicho.

## Fluxo de uso
1. Ler `references/fontes-brasil.md` para alinhar contexto de mercado e boas práticas.
2. Definir 5 parâmetros antes de escrever qualquer prompt:
   - público local (cidade/faixa de renda)
   - oferta principal (procedimento/benefício)
   - objeção dominante (preço, medo, tempo, confiança)
   - CTA principal (WhatsApp único)
   - limite de promessa (sem resultados garantidos)
3. Produzir mensagens no padrão de 3 blocos:
   - abertura humana curta
   - 1 pergunta de qualificação
   - micro-CTA (próximo passo simples)
4. Revisar com checklist de qualidade (seção final).

## Padrão de escrita (copiar e aplicar)
- Escrever frases curtas (máx. 18 palavras por frase).
- Usar vocabulário brasileiro cotidiano, sem "tradutês".
- Evitar jargão técnico do nicho na primeira resposta.
- Fazer **uma pergunta por vez**.
- Confirmar contexto antes de oferecer preço.
- Direcionar para WhatsApp apenas após sinal mínimo de intenção.

## Template de resposta para DM
Use este molde e adapte para cada oferta:

```txt
Perfeito, [nome]. Entendi que você quer [objetivo principal].

Pra te indicar o melhor caminho, me diz: você busca isso para começar ainda este mês?

Se preferir, já te passo no WhatsApp os próximos passos e valores iniciais. Quer?
```

## Padrão de objeções
- Preço: "Faz sentido olhar investimento. Antes, te explico qual opção encaixa melhor no seu objetivo."
- Medo/insegurança: "Totalmente normal essa dúvida. A avaliação/consulta existe justamente para indicar com segurança o que vale para seu caso."
- Falta de tempo: "Sem problema, te mando um resumo objetivo e você vê no melhor horário."

## Checklist final (go/no-go)
Só publicar prompt se todos os itens forem "sim":
- Soa como brasileiro real falando com respeito?
- Tem pergunta de qualificação explícita?
- Evita promessa absoluta de resultado?
- Encaminha para WhatsApp com CTA simples?
- Mantém tom consultivo (não agressivo)?
